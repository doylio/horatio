Shakespear.io is the easiest way to access the works of William Shakespeare. No bells and whistles, just the works of the great writer based on your query. Try it out now!

API Request Types

Text Request

http://www.shawnrobertdoyle.com/p/shakespear-io/text/[play]/[act]/[scene]/[lines]

A text request returns all text in the Complete Works of Shakespeare that match your request.

Example Request
GET http://www.shawnrobertdoyle.com/p/shakespear-io/text/hamlet/3/1/64


{
  "URL": "/text/hamlet/3/1/64",
  "time": "Tue Dec 25 2018 01:59:03 GMT+0000 (UTC)",
  "data": [
    {
      "play_name": "Hamlet",
      "play_id": 30,
      "play_key": "hamlet",
      "play_text": [
        {
          "act": 3,
          "scene": 1,
          "text": [
            {
              "character": "HAMLET",
              "character_id": 323,
              "lines": [
                {
                  "line_no": 64,
                  "line": "To be, or not to be: that is the question:"
                }
              ]
            }
          ]
        }
      ]
    }
  ]
}
Request Parameters

Text location parameters slot directly into the URL. Filter parameters should be specified as HTTP query parameters. Leave off all parameters to receive the entire collection.

Location Parameters
  • play number or key
    The play you want to receive data from. Can be inputted as the numeric play_id, or the play_key, shorthand version of the name of the play.
  • act number
    The act you want to target.
  • scene number
    The scene you want to target.
  • lines number or range
    The line or lines you would like to receive. Can by inputed as a single line /64, or as a range of lines /34-72.
Filter Parameters

Inputted as HTTP query parameters.

  • char number or name
    Filters the lines in the text location where the character matches this value. Can be inputted as the numeric character_id, or by the name of the character.
  • search text
    A fuzzy search of the lines in the text location. Returns lines ordered by how closely they match this value.

Play Request

http://www.shawnrobertdoyle.com/p/shakespear-io/play/[play]

A play request returns information about a single play, or a whole list.

Example Request
GET http://www.shawnrobertdoyle.com/p/shakespear-io/plays/macbeth


{
  "URL":"/play/macbeth",
  "time": "Tue Dec 25 2018 01:59:03 GMT+0000 (UTC)",
  "data":[
    {
      "full_name":"Macbeth",
      "id":33,
      "key":"macbeth",
      "year_published":1623
    }
  ]
}
Request Parameters

Receive the list of plays, optionally filtered by the following parameters.

  • play number or key
    Returns only the information about that specific play.

Character Request

http://www.shawnrobertdoyle.com/p/shakespear-io/character/[play]?char=[char]

A character request returns information about characters.

Example Request
GET http://www.shawnrobertdoyle.com/p/shakespear-io/character/macbeth?char=duncan


{
  "URL": "/p/shakespear-io/character/macbeth?char=duncan",
  "time": "Mon Dec 24 2018 23:27:04 GMT+0000 (UTC)",
  "data": [
    {
      "name": "DUNCAN",
      "id": 80,
      "play": {
        "play_name": "Macbeth",
        "play_id": 33,
        "play_key": "macbeth"
      }
    }
  ]
}
Request Parameters

Receive the list of characters, optionally filtered by the following parameters. If requested without parameters, it returns a list of all characters in the canon.

  • play number or key
    Returns the list of characters from the specified play.
  • char number or name
    Filters the character list for those that matches this value. Can be inputted as the numeric character_id, or by the name of the character. Inputed as a query string.
